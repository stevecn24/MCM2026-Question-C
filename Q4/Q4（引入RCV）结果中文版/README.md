# 《与星共舞》优化评分系统分析结果

## 概述
本文件夹包含对《与星共舞》评分系统的优化分析结果，引入了时间演化的加权模型和RCV（排序复选制）偏好模型。

## 优化特性
- **时间演化加权模型**: 考虑历史数据的时效性，近期数据权重更高
- **缓启动模型**: 前3周权重较低，避免初期波动影响
- **RCV偏好模型**: 将粉丝投票转化为排序得分，更准确反映偏好
- **指数衰减**: 使用指数函数计算时间权重
